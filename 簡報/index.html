AI 建置到班打卡系統

曾慶良（阿亮老師）| 3A 科技研究室

這一章你會做出什麼

🗄️

三頁面打卡系統

首頁導覽、管理介面、打卡介面，Flask + SQLite CRUD。

📑

CSV 匯入匯出

名單匯入、報表匯出，處理繁中編碼（UTF-8 + BOM）。

📷

照片自動配對

用「班級+座號」命名，自動對應學生照片。

🔄

Fetch 非同步

前端 Fetch × 後端 JSON，不刷新頁面就更新。

🎯 核心觀念：不是所有系統都需要 AI。打卡是結構化資料操作，用資料庫最合適——這是全書的「回歸基礎」章。

STEP 1下載專案 & 安裝套件

# 下載專案 git clone https://github.com/ChatGPT3a01/AI-Attendance-System.git cd AI-Attendance-System # 安裝套件（超輕量，只有 Flask 和 Werkzeug） pip install -r requirements.txt Successfully installed Flask-2.3.3 Werkzeug-2.3.7 ← 成功
🪶 這章相依超輕量、完全本機運作：不需 API Key、不需網路、沒有雲端費用。

STEP 2啟動系統

python app.py * Running on http://localhost:5000 ← 啟動成功

打開瀏覽器前往 http://localhost:5000，會看到三個入口：首頁 / 管理介面 / 打卡介面。

⚠️ 找不到 python？Windows 試 py app.py；Mac 試 python3 app.py。

STEP 3匯入學生名單（CSV）

  1. 進「管理介面」，點「下載 CSV 模板」。
  2. 用 Excel 打開模板，填入班級、座號、姓名等欄位。
  3. 存檔（另存為 UTF-8 編碼，避免亂碼）。
  4. 回到管理介面「選擇 CSV 檔案」→ 上傳。
⚠️ 繁中亂碼？模板已含 BOM 標記 讓 Excel 正確顯示；若仍亂碼，用記事本開啟另存 UTF-8。照片檔名用「班級+座號」就會自動配對。

STEP 4打卡 & 匯出報表

  1. 進「打卡介面」，學生點自己的名字/座號即完成打卡（Fetch 即時寫入 SQLite，不刷新頁面）。
  2. 回「管理介面」可依日期範圍查詢出缺勤。
  3. 點「匯出報表」下載 CSV（UTF-8，可直接用 Excel 開）。
🎉 一套「名單匯入 → 打卡 → 報表匯出」的到班系統，完全跑在你自己的電腦上！
